## 📖 About This Repository

This repository contains **complete, well-organized study notes** for the subject **Data Visualization and Analytics (DVA)** — Subject Code **AL-603-B** — as taught in the **B.Tech Artificial Intelligence & Machine Learning (AI & ML)** program at **Rajiv Gandhi Proudyogiki Vishwavidyalaya (RGPV)**, Bhopal.

The notes are structured **unit-wise** in downloadable **PDF format**, covering the entire syllabus from foundational data definitions and statistical concepts to advanced data analysis, data wrangling, data ecosystems, and data visualization tools.

Whether you're preparing for **semester exams**, **internal assessments**, or simply strengthening your understanding of data analytics and visualization concepts — this repository has you covered.

## 📋 Syllabus Overview

| Unit | Title | Key Topics |
|:----:|:------|:-----------|
| 1 | [Data Definitions, Analysis Techniques & Statistical Concepts](Unit-1-Data-Definitions-Analysis-Techniques-and-Statistical-Concepts/) | Types of data, data collection methods, descriptive statistics, probability distributions, hypothesis testing, inferential statistics |
| 2 | [Advanced Data Analysis Techniques](Unit-2-Advanced-Data-Analysis-Techniques/) | Regression analysis, classification, clustering, time series analysis, predictive modeling, advanced analytics methods |
| 3 | [Data Wrangling](Unit-3-Data-Wrangling/) | Data cleaning, data transformation, handling missing data, data integration, data quality assessment, ETL processes |
| 4 | [Data Ecosystem](Unit-4-Data-Ecosystem/) | Data infrastructure, data lakes, data warehouses, big data platforms, data governance, data pipelines, cloud data services |
| 5 | [Data Visualization Tools](Unit-5-Data-Visualization-Tools/) | Visualization libraries, Tableau, Power BI, Matplotlib, Seaborn, interactive dashboards, chart types, storytelling with data |
