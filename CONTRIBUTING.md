# 🤝 Contributing to DVA Notes (AL-603-B)

Thank you for your interest in contributing to this repository! Every contribution — whether it's fixing a typo, adding notes, or improving the structure — helps fellow students.

---

## 📋 How Can You Contribute?

### 🐛 Report Errors
If you find any mistakes in the notes:
1. Open an [Issue](https://github.com/Vivek30092/Data-Visualization-and-Analytics-Notes-AL603B/issues)
2. Describe the error clearly (mention the unit and page number if applicable)
3. Suggest the correction if you know it

### 📝 Add Notes, PYQs, or Solved Papers
Have supplementary notes, previous year question papers, or solved papers in **PDF format**?
1. Fork this repository
2. Add your PDF files to the appropriate folder:
   - **Unit notes** → Place in the relevant `Unit-X-...` folder
   - **PYQs / Question papers** → Place in the `PYQs/` folder
   - **Solved papers** → Place in the relevant unit folder or `PYQs/` folder
3. Use descriptive file names (e.g., `DVA-Unit-3-Additional-Notes.pdf`, `al-603-b-dva-dec-2024.pdf`)
4. Submit a Pull Request with a clear description of what you added

### 🔗 Share Useful Links
Know of helpful articles, videos, or tutorials related to Data Visualization and Analytics?
- Open an Issue with the link and a brief description
- Or add them to the relevant unit's README

### 💡 Suggest Improvements
Ideas to improve the repository structure, content, or documentation? Open an Issue and let us know!

---

## 📐 Guidelines

- **Be respectful** — Follow the [Code of Conduct](CODE_OF_CONDUCT.md)
- **Keep it relevant** — All contributions should be related to Data Visualization and Analytics (AL-603-B)
- **Use clear file names** — Follow the existing naming convention: `DVA-Unit-X-Description-AL603B.pdf`
- **Write good commit messages** — Describe what you changed and why
- **Test your links** — Make sure any Markdown links you add are working

---

## 🚀 Steps to Contribute

```bash
# 1. Fork this repository (click the Fork button on GitHub)

# 2. Clone your fork
git clone https://github.com/YOUR-USERNAME/Data-Visualization-and-Analytics-Notes-AL603B.git

# 3. Create a new branch
git checkout -b add-unit3-extra-notes

# 4. Make your changes and commit
git add .
git commit -m "Add supplementary notes for Unit 3 Data Wrangling"

# 5. Push to your fork
git push origin add-unit3-extra-notes

# 6. Open a Pull Request on GitHub
```
