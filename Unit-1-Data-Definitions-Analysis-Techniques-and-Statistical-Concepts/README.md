## 📋 Topics Covered

- Introduction to data and data types
- Qualitative vs. quantitative data
- Data collection methods and sources
- Exploratory data analysis (EDA)
- Descriptive statistics — mean, median, mode, variance, standard deviation
- Probability fundamentals and distributions
- Normal distribution and its properties
- Hypothesis testing basics
- Inferential statistics introduction
- Sampling techniques and sampling errors
- Correlation and covariance
- Data analysis workflows and best practices

---

## 🎯 Learning Objectives

After studying this unit, you should be able to:

- Define and classify different types of data used in analytics
- Apply descriptive statistical measures to summarize datasets
- Understand probability distributions and their role in data analysis
- Perform basic hypothesis testing
- Differentiate between descriptive and inferential statistics
- Select appropriate analysis techniques for different data types

---

## 📝 Description

Unit 1 lays the foundation for the entire Data Visualization and Analytics course. It introduces students to the core definitions and classifications of data, essential analysis techniques, and fundamental statistical concepts. This unit is critical for building the analytical mindset needed for advanced topics in subsequent units — from data wrangling and ecosystem concepts to visualization tools.
