## 📋 Topics Covered

- Introduction to data wrangling and its importance
- Data cleaning techniques — handling noise, outliers, and inconsistencies
- Handling missing data — imputation, deletion, and interpolation methods
- Data transformation — normalization, standardization, encoding
- Data integration from multiple sources
- Data quality assessment and metrics
- ETL (Extract, Transform, Load) processes
- Data formatting and reshaping
- String manipulation and text cleaning
- Working with structured and unstructured data
- Data validation and verification
- Tools and libraries for data wrangling (Pandas, NumPy)

---

## 🎯 Learning Objectives

After studying this unit, you should be able to:

- Identify and resolve common data quality issues
- Apply data cleaning techniques to real-world messy datasets
- Handle missing values using appropriate imputation strategies
- Transform and normalize data for analysis
- Integrate data from heterogeneous sources
- Design and implement ETL workflows
- Use data wrangling tools and libraries effectively

---

## 📝 Description

Unit 3 focuses on the critical preprocessing step that makes raw data ready for analysis and visualization. Data wrangling — also known as data munging — is one of the most time-consuming yet essential steps in any data analytics pipeline. This unit equips students with practical techniques to clean, transform, and prepare data, ensuring the accuracy and reliability of all downstream analysis.
