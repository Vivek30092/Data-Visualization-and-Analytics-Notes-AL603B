## 📋 Topics Covered

- Regression analysis — linear and multiple regression
- Classification techniques and algorithms
- Clustering methods — K-means, hierarchical clustering
- Time series analysis and forecasting
- Predictive modeling fundamentals
- Association rule mining
- Dimensionality reduction techniques (PCA)
- Model evaluation and validation
- Advanced statistical methods for analytics
- Feature selection and engineering
- Overfitting, underfitting, and model optimization

---

## 🎯 Learning Objectives

After studying this unit, you should be able to:

- Apply regression models to analyze relationships in data
- Implement classification and clustering techniques
- Perform time series analysis for trend detection and forecasting
- Build and evaluate predictive models
- Understand the trade-offs between different analysis methods
- Select appropriate advanced techniques for real-world data problems

## 📝 Description

Unit 2 builds on the statistical foundation from Unit 1 and dives into advanced analytical methods. Students learn powerful techniques like regression, classification, clustering, and predictive modeling — skills essential for extracting actionable insights from complex datasets. These concepts are widely applicable across data science, business analytics, and AI/ML domains.
